MyOSU is Undergoing Scheduled Maintenance

The MyOSU portal is currently being upgraded and is unavailable at this time.

Other key administrative systems will also be undergoing maintenance over Labor Day weekend. All applications will be returned to service by 11:59PM on Monday, September 7.

Please see the network annoucement for more information.

Popular links avilable through maintenance period:

  • Blackboard
  • Canvas / Learn@OregonState
  • fsaAtlas
  • Google Drive
  • MyTime/EmpCenter
  • ONID Email
  • Online Recruitment and Position Descriptions
  • Scholarship Management System

The following links may be unavailable or have limited functionality during maintenance

  • Appworx
  • Banner
  • CORE Reports
  • Classrooom Availability (WebViewer)
  • Foundation Direct Payment
  • Foundation Reimbursement System
  • Grant Reporting
  • My Degrees: Access through Online Services
    • Students: Select Student Records from the main menu, select MyDergrees, then select Submit MyDegrees
    • Advisors: Select the Faculty & Advisors tab, select MyDegrees, then select Submit MyDegrees
  • Nolij
  • Online Services
  • OSCAR
  • TouchNet Marketplace
  • TRES: Please use Online Services and then select TRES from the main menu
  • Workflow

Grades will not be accessible during administrative system maintenance.

When available, access grades as follows:

  • Launch Online Services
    • Students: Select Student Records from the main menu, select Final Grades, select term from Select a Term, then click Submit.
    • Faculty & Advisors: Select the Faculty & Advisors tab, select Final Grades Menu, then select one of the following options:
      • Final Grades - Keyed Entry
      • Final Grades - File Upload
      • Final Grades - Upload from Blackboard
      • View Missing Final Grades
